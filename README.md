# OI-Team-Learning-Data-System

OI:Olympics in Informations，即信息学团队知识系统。

本系统来自潮州市金山中学信息学团队。

## 针对对象

本知识系统为信息学团队服务，尽量提供全面的学习资料，供团队课上资料运用及课后自习用

学习资料由制作人员制作而成，耗费了一定的心血，成品将密封保存在网盘中。

下一届的成员大可直接fork这个项目直接接手。
